## 🌐 Просмотр документации

Документация доступна в виде веб-сайта по адресу: **http://localhost/docs**

### Особенности веб-версии:
- 📱 Адаптивный дизайн для мобильных устройств
- 🎨 Подсветка синтаксиса для блоков кода
- 📑 Боковая навигация по всем документам
- 🔍 Поиск по документации (через браузер)

## 📝 Формат файлов

Все документы написаны в формате **Markdown** с поддержкой:
- Таблиц (GitHub Flavored Markdown)
- Блоков кода с подсветкой синтаксиса
- Заголовков различных уровней
- Списков и цитат
- Ссылок и изображений

## 🔧 Добавление новой документации

1. Создайте `.md` файл в соответствующей директории `docs/`
2. Добавьте заголовок первого уровня (`# Заголовок`)
3. Документ автоматически появится в веб-интерфейсе
