# RoadCheck UI

Frontend административной панели RoadCheck на React, Vite и Mantine.

## Реализовано

- авторизация и разграничение доступа по ролям;
- аналитический дашборд с обращениями, погодой и зонами риска;
- карта опасных участков и тепловой слой;
- страницы обращений, регионов, дорог, пользователей и опасных зон;
- раздел прогнозов и рекомендаций для модератора;
- API-клиенты и тесты для основных интеграций.

## Запуск

```bash
npm install
npm run dev
```

Адрес backend задается переменной `VITE_API_BASE_URL` в `.env`.

## Команды

```bash
npm run build
npm run lint
npm test
```
